# RMOTR Docker Images

Run Jupyter labs in docker:

```bash
docker run -p 5000:8888 -it data-science-python-3.6 jupyter lab --i 0.0.0.0 --port=8888 --no-browser --allow-root
```

* https://hub.docker.com/r/waleedka/modern-deep-learning/~/dockerfile/
